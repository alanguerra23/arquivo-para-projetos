# Estrutura de Pastas do Projeto

```js
├── .obsidian/ `Pasta do aplicativo obsidian para escrever markdown.`
├── .vscode `Pasta dos arquivos de configuração do editor de código vscode.`
│   └── workspace.code-workspace `Arquivo de configuração do workspace do vscode.`
├── .github/ `Contém os arquivos do github.`
├── docs/ `Contém arquivos úteis com informações sobre o projeto.`
├── images/ `Contém arquivos de imagens e svg.`
```
